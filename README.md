# Go File Handle
[![My Skills](https://skillicons.dev/icons?i=go)](https://github.com/hosseinyn/GoFileHandle)

A simple file handling project in golang. This project uses OS, IO, IO/IOUTIL, Bufio , Strings and fmt libraries to perform handling files and handling user inputs in the go programming language.


------------

### Features
- Create file
- Check is file exists / Is file empty
- Write in a file
- Read file line by line
- Delete file
- Rename file
- Copy file contents to another file
- Information about file (directory or regular , permissions)
- List all files and directories in a directory with their information

## How to use?
1. Clone the repository
``git clone https://github.com/hosseinyn/GoFileHandle.git``

2. Open project directory
`cd GoFileHandle`

3. Run the file using `go` command :
`go run files.go`
or
Run the file using `files.exe` in windows

4. Or build the file:
`go build files.go`
